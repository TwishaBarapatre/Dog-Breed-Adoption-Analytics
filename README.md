**Dog Breed Adoption Analytics** is a data analytics project that analyzes dog adoption trends using Python, web scraping, data cleaning, statistical analysis, and Power BI dashboards.

The project collects dog adoption data from online pet adoption platforms, transforms raw data into structured datasets, and generates breed-wise and location-wise adoption insights. The final results are visualized through interactive dashboards to better understand adoption behavior and trends.

**Tech Stack:** Python, Pandas, BeautifulSoup, Requests, Power BI

**Project Workflow:**
Data Collection → Data Cleaning → Data Analysis → Dashboard Visualization

**Generated Datasets:**

* `dogs_raw.csv`
* `dogs_cleaned.csv`
* `breed_adoption_stats.csv`
* `location_adoption_stats.csv`

**Key Insights:**

* Fastest and slowest adopting breeds
* Location-based adoption trends
* Age and size impact on adoption speed
* Overall adoption behavior patterns
